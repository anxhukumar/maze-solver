# Maze Solver

A simple Python maze generator and solver with a Tkinter visualization.

## Features
- Generates a random maze using recursive backtracking.
- Solves the maze using depth-first search.
- Animates both generation and solving in a window.

## Requirements
- Python 3.10+ (or any recent Python 3 version)
- Tkinter (usually included with standard Python installations)

## Run
```bash
python main.py
```

## Run tests
```bash
python tests.py
```

## Project files
- `main.py` — app entry point.
- `maze.py` — maze generation and solving logic.
- `cell.py` — cell model and drawing moves.
- `graphics.py` — basic Tkinter drawing primitives.
- `tests.py` — basic unit tests.
